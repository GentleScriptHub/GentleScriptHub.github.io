# GentleHub Website

This is the source code for your GentleHub script site.

## Setup Instructions:
1. Upload all these files to a GitHub repository.
2. Go to Settings > Pages and set the source to "main" branch, root (/).
3. Visit your free live site at: https://yourusername.github.io/repo-name/

You can customize:
- Discord embed
- Styles (in style.css)
- Features and text (in index.html)
